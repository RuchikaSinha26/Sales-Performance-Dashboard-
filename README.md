# 📊 Sales Performance Dashboard

## 📌 Overview
An interactive Power BI dashboard built using the SuperStore Sales Dataset to analyze sales performance, profitability, customer behavior, and regional trends. The dashboard enables stakeholders to monitor key business metrics and make data-driven decisions through dynamic visualizations and KPI tracking.

## 🎯 Problem
Businesses often struggle to track revenue, profit, orders, and sales performance across different regions and product categories due to fragmented reporting and manual analysis.

## 🔍 Analysis
- Processed and analyzed 10,000+ sales records using Excel and Power BI.
- Performed data cleaning, transformation, and modeling using Power Query.
- Created DAX measures for KPI calculations and business insights.
- Analyzed sales, profit, orders, customer segments, product categories, and regional performance.

## 📈 Insights
- Identified top-performing products and profitable regions.
- Highlighted underperforming products and low-profit areas.
- Analyzed customer segment contribution to overall revenue.
- Evaluated sales trends and payment mode preferences.

## 🚀 Impact
- Automated reporting workflows, reducing report preparation time by approximately 30%.
- Improved business visibility through real-time dashboards.
- Enabled faster and more informed decision-making.
